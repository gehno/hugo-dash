# hugo-dash
fully static link collection dashboard created with hugo and tailwind

## Used Technologies

- Hugo
- Tailwind CSS
- Font Awesome
- Docker

## Development

### Setup

```shell
npm install
```

### Run Local Server

```shell
./run.sh
```

## Installation

### docker-compose

There is a [minimal](/docker-compose/minimal/docker-compose.yaml) and a [nginx](/docker-compose/nginx/docker-compose.yaml) configuration.
